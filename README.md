# ✅ Todo List App

> *Stay organized. Get things done.*

A beautiful, feature-rich task manager built with vanilla HTML, CSS & JavaScript — with smooth animations, local storage persistence, and a clean dark UI.

🔗 **[Live Demo](#)*https://manishabaloda4-cloud.github.io/to-do-app/*

---

## ✨ Features

- ➕ **Add tasks** — type and press Enter or click Add
- ✅ **Complete tasks** — animated custom checkbox
- ✏️ **Inline editing** — double-click any task to edit it directly
- 🗑️ **Delete tasks** — hover to reveal delete button with animation
- 🔍 **Filter** — view All / Active / Completed tasks
- 📊 **Progress bar** — visual completion tracker
- 🧹 **Clear completed** — remove all done tasks at once
- 💾 **Auto-save** — tasks saved to localStorage, survive page refresh
- 📅 **Today's date** — displayed in the header
- 🎨 **Smooth animations** — tasks slide in/out with spring physics

## 🎯 How to Use

| Action | How |
|--------|-----|
| Add task | Type in input → press **Enter** or click **Add** |
| Complete task | Click the **circle checkbox** |
| Edit task | **Double-click** the task text |
| Delete task | **Hover** over task → click trash icon |
| Filter tasks | Click **All / Active / Done** buttons |
| Clear done | Click **Clear completed** at the bottom |
